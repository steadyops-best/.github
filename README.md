# SteadyOps

SteadyOps provides senior DevOps/SRE support for teams that need reliable production systems, safer deployments, monitoring, incident response and recovery runbooks.

## What SteadyOps focuses on

- Production reliability
- CI/CD and rollback safety
- Kubernetes production readiness
- PostgreSQL HA and backup/restore review
- Monitoring and observability
- Disaster recovery runbooks
- nginx, Docker, systemd and Linux operations
- Infrastructure audits and automation
